# Adds simple vanilla-like plates to Minecraft

## Features:
- **Variants to come**
- **Crafted with a bowl**
- **Can hold up to 1 stack of 1 item (will increase)**
- **Can be interacted with by right-clicking**
- **Can hold any type of item**
- **Plate.**

You may use this mod freely in your modpack as long as it's credited.

Versions ports may come later (mainly Fabric and NeoForge 1.21.1, and maybe 1.21.11 if I got time).

Credits to Mojang for the current Plate's texture.
